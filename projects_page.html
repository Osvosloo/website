    • Game Development

    Description: I made a small PC game for my friends and I to play. I grew tired of the industry-standard for games and wanted to have control over the core gameplay, which isn't something that all game studios offer players. So, I decided to use the Unity Game Engine for the project. The core gameplay is a strategy/luck-based card game, similar to games like 'Magic: The Gathering' or 'GWENT', the card game from 'The Witcher', where players can collect cards by playing the game but must also use some randomly drawn common cards during a match. Up to four players can play the game. Even though it's a bit rough around the edges, it was still a fun project.

    *Technologies Used:

  • The Unity Game Engine
  • C#.NET
  • ProtonEngine Game Servers
